# 阶段 6：嵌入式开发环境

## 目标

搭建基于 Keil MDK 和 STM32CubeMX 的嵌入式开发环境，掌握 ARM Cortex-M 项目创建、编译、烧录与调试流程。

## 前置依赖

- 阶段 3（C 语言、CMake 与 OpenCV）
- 阶段 1（Git）

## 任务

| 子项 | 任务 | 验收 |
|------|------|------|
| 6.1 Keil MDK | 下载 [Keil MDK](https://www.keil.com/fid/armkeil/files/eval/mdk543.exe) 并完成安装；通过 License Management 注册；确认编译器可用 | Keil uVision 可正常启动；新建空项目编译通过 |
| 6.2 STM32CubeMX | 下载 [STM32CubeMX](https://www.st.com/en/development-tools/stm32cubemx.html) 并安装；选择一个 STM32 型号生成初始化 C 代码；在 Keil 中打开生成的项目 | CubeMX 成功生成 Keil 项目；可在 uVision 中编译 |
| 6.3 编译与烧录 | 在 Keil uVision 中配置烧录工具；编译项目并下载到开发板 | 开发板运行用户程序 |
| 6.4 调试入门 | 在 Keil 中设置断点；启动调试会话；单步执行、查看寄存器与变量值 | 断点可命中；可查看变量实时值 |

## 快速参考

```
# Keil MDK 安装
1. 下载 mdk543.exe 并运行安装
2. 安装完成后打开 uVision
3. 通过 License Management 注册

# STM32CubeMX 生成初始化代码
1. 新建项目，选择 MCU 型号
2. 配置时钟、GPIO、外设
3. Project Manager 中 Toolchain 选择 MDK-ARM V5
4. 生成代码，在 Keil 中打开

# Keil 编译、烧录与调试
1. Project → Build Target  # 编译
2. Flash → Download         # 烧录到开发板
3. Debug → Start/Stop Debug Session  # 启动调试
```
