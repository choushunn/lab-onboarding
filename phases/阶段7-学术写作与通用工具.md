# 阶段 7：学术写作与通用工具

## 目标

掌握 Markdown 和 LaTeX 两种写作工具的使用，能够撰写格式规范的文档、论文和技术报告。

## 前置依赖

- 阶段 2（Python 与开发环境）
- 阶段 1（Git）

## 任务

| 子项 | 任务 | 验收 |
|------|------|------|
| 7.1 Markdown 基础 | 学习 Markdown 语法：标题、列表、表格、代码块、链接、图片、公式；在 VSCode 中创建 `.md` 文件并使用预览；在 GitHub 仓库中使用 Markdown 编写 README | 能独立编写一个包含表格、代码块、链接的 Markdown 文档 |
| 7.2 LaTeX 安装与配置 | 安装 [TeX Live](https://tug.org/texlive/)；在 VSCode 中安装 LaTeX Workshop 扩展；创建一个 `.tex` 文档，编译为 PDF | `xelatex` 或 `pdflatex` 编译成功，输出 PDF 文件 |
| 7.3 LaTeX 论文结构 | 学习 LaTeX 文档结构：`documentclass`、`begin{document}`、`section`、`subsection`；插入图片、表格、参考文献；编译含参考文献的文档 | 完成一篇含标题、章节、图片、表格、参考文献的示例论文 |
| 7.4 VSCode + LaTeX 工作流 | 在 VSCode 中配置 LaTeX Workshop 实现保存自动编译；使用 `latexmk` 连续编译 | 修改 `.tex` 后保存自动刷新 PDF |

## 快速参考

```bash
xelatex paper.tex
bibtex paper
xelatex paper.tex
xelatex paper.tex

latexmk -xelatex paper.tex
```
